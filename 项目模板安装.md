#### 如何安装模板

```shell
dotnet new install .\EmpTechAvalonia.1.0.0.nupkg
```

![image-20251205152422634](https://raw.githubusercontent.com/zl-aric/MyImage/main/images/image-20251205152422634.png)

#### 如何通过模板创建项目

```shell
dotnet new EmpTechAvalonia -n Emp2931A2
```

![image-20251205152607034](https://raw.githubusercontent.com/zl-aric/MyImage/main/images/image-20251205152607034.png)

#### 如何卸载模板

```shell
dotnet new uninstall EmpTechAvalonia
```
